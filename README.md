# Sentimental-Analysis-using-Twitter-Dataset
To perform Sentimental Analysis on Twitter Dataset.

<br/>

## Objectives
* To analyze thousands of tweets and try to predict people's sentiment.
* To build a Machine Leaning model that take tweets as input and give us the sentiment
of those as output.

<br/>

## Software/ Libraries Used
Python, Jupyter Notebook/Kaggle Notebook, Pandas, Numpy, Seaborn, Matplotib, nlkt
